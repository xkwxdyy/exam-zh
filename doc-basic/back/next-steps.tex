\subsection{高级功能列表}

本入门文档未涵盖的高级功能（完整文档中有详细说明）：

\subsubsection{高级题型}

\begin{itemize}
  \item \textbf{选择标记题}：在选项前打钩或打叉
  \item \textbf{连线题}：用线段连接左右两列
  \item \textbf{计算题排版}：专门的计算题环境
\end{itemize}

\subsubsection{高级排版}

\begin{itemize}
  \item \textbf{图文混排}：文字和图片的灵活排列
  \item \textbf{试卷合集}：一个文档包含多套试卷
  \item \textbf{自定义题号样式}：使用各种编号格式
  \item \textbf{密封线详细配置}：位置、样式、文字等
\end{itemize}

\subsubsection{字体和样式}

\begin{itemize}
  \item \textbf{西文字体设置}：Times、Latin Modern等
  \item \textbf{数学字体设置}：XITS、STIX等
  \item \textbf{符号定制}：中国化数学符号
\end{itemize}


\subsection{LaTeX 进阶学习}

掌握更多 \LaTeX{} 知识可以更灵活地使用 exam-zh。

\subsubsection{推荐资源}

\begin{description}
  \item[lshort-zh-cn] \LaTeX{} 入门教程（中文）
    \begin{itemize}
      \item 命令行：\cmd{texdoc lshort-zh-cn}
      \item 在线：\url{https://ctan.org/pkg/lshort-chinese}
    \end{itemize}

  \item[LaTeX Stack Exchange] 全球最大的 \LaTeX{} 问答社区
    \begin{itemize}
      \item \url{https://tex.stackexchange.com/}
      \item 搜索问题通常能找到答案
    \end{itemize}

  \item[CTeX 论坛] 中文 \LaTeX{} 社区
    \begin{itemize}
      \item \url{https://www.latexstudio.net/}
      \item 适合中文用户交流
    \end{itemize}
\end{description}

\subsubsection{学习重点}

对于使用 exam-zh 的用户，建议重点学习：

\begin{enumerate}
  \item \textbf{数学公式}：\pkg{amsmath} 宏包的使用
  \item \textbf{表格制作}：\pkg{tabularray} 或 \pkg{tabular}
  \item \textbf{图片插入}：\pkg{graphicx} 宏包
  \item \textbf{页面设置}：\pkg{geometry} 宏包
  \item \textbf{宏包管理}：如何查找和使用其他宏包
\end{enumerate}
